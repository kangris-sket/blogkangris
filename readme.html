Kami selalu siap untuk tantangan baru. Yang ini ditujukan kepada kami oleh  + Ben Morrell  yang ingin mengubah blognya menjadi sebuah buku, atau lebih tepatnya dokumen Word yang bisa diedit sebelum dikirim ke printer. Tentu, dia bisa saja pergi dan menyalin / menyisipkan setiap posting blog ke Word, tetapi blognya dimulai pada tahun 2002, setelah lebih dari 10 tahun posting blog.

Namun kami berhasil membuat proses transformasi ini menggunakan alat yang mungkin dimiliki semua blogger.

Langkah 1: Mendapatkan ekspor konten blog Anda
Cara terbaik untuk mendapatkan semua konten Anda dari Blogger adalah dengan menggunakan situs yang disebut Front Pembebasan Data . DL adalah tim teknisi di dalam Google yang tujuannya adalah untuk memudahkan pengguna memindahkan data mereka keluar masuk Produk Google (jadi aman!). Buka di sini untuk mengambil konten Anda dari salah satu blog Anda (dengan mengklik Konfigurasi kemudian Buat Arsip). Anda akan menerima file ZIP dari ekspor ini yang berisi file di dalamnya seperti YourBlog.atom.

Langkah 2: Ubah nama file
Buka file ZIP dan seret file YourBlog.atom ke folder di desktop Anda. Ubah nama file dari YourBlog.atom ke YourBlog.xml

Langkah 3: Unduh lembar gaya dan
Unduh Template xsllembar gaya ini dan template XSL ini dan simpan di direktori yang sama dengan file YourBlog.xml Anda yang baru.

Langkah 4: Edit satu baris dalam file XML di editor teks
Buka file XML di editor teks dan ubah:
<? Xml-stylesheet href = "http://www.blogger.com/styles/atom.css" jenis = "text / css"?>
menjadi
<? xml-stylesheet href = "transform.xsl" type = "text / xsl"?>

Langkah 5: Seret file XML ke browser Safari atau Internet Explorer
Ya, browser Safari atau browser Internet Explorer. Chrome dan Firefox mungkin mencoba menerapkan gaya bawaan mereka sendiri ke file XML, sehingga berfungsi di dua browser lainnya.


File XSL yang telah kami siapkan untuk melakukan transformasi berikut:
<h1> Judul postingan </ h1>
<h2> Penulis postingan </ h2>
<h3> Tanggal posting </ h3>
Konten posting blog

Ini melakukan ini dalam urutan terbalik, jadi pesan ini beralih dari pos Lama ke Terbaru. Jika Anda mau, Anda dapat mengubah ini menjadi naik dalam file XSL, dan Anda dapat menambahkan lebih banyak ke file XSL jika Anda ingin memasukkan komentar, tetapi kami menganggap ini cukup untuk sebagian besar. Ini juga akan memuat gambar eksternal untuk posting blog.


Langkah 6: Pilih semua dan copy paste ke Word
Ini mungkin agak sulit tergantung pada seberapa besar blog Anda, tetapi kami mampu melakukan arsip blog 12 meg dengan mudah. Cukup pilih semua, salin, kemudian buka dokumen Word baru dan tekan tempel. Anda mungkin perlu menunggu sebentar, tetapi itu akan berhasil.

Langkah 7: Simpan dokumen Word
Simpan dokumen sebelum Anda mulai mengerjakannya! Setelah disimpan, Anda dapat mengedit isi hati Anda, mengubah ukuran gambar, memindahkannya, memformat ulang ... semuanya. Tautan dan gambar semuanya diawetkan serta banyak formatnya.


Dan itu dia! Jika Anda memiliki pertanyaan, silakan tinggalkan komentar di bawah ini!
